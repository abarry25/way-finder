# i-lab Program Fit Guide

A standalone HTML page summarizing who each Harvard Innovation Labs pilot program is designed for, who gets the most out of it, and who it isn't right for.

Built as a reference artifact to support the development of the i-lab's wayfinding tool.

---

## What's in this file

`ilab_program_fit_guide.html` covers two summer 2026 pilot programs:

- **Customer Validation Primer** — a single-session workshop on gathering and interpreting customer feedback
- **Customer Discovery Sprint** — a 3-week structured program combining workshops, real interviews, peer reflection, and advisor feedback

For each program it documents three tiers:

| Tier | What it answers |
|------|----------------|
| Fit for | People who could broadly benefit |
| Best fit for | People where this will make a meaningful difference right now |
| Not fit for | People this would waste time for |

A sequencing note at the bottom flags that the Primer is a useful prerequisite mindset for the Sprint.

## How to use it

**View in browser**
Open `ilab_program_fit_guide.html` in any modern browser. Fonts and icons load from Google Fonts and jsDelivr CDN — an internet connection is required.

**Screenshot for ClickUp or slides**
Open in Chrome, resize to your preferred width, and screenshot. The layout is optimized for ~860px wide.

**Embed in ClickUp**
Host the file (e.g. via GitHub Pages) and paste the URL into a ClickUp `/embed` block to render it inline in a doc.

**Host on GitHub Pages**
1. Push this file to your repository
2. Go to Settings → Pages
3. Set branch to `main`, folder to `/ (root)`
4. Your file will be live at `https://yourusername.github.io/your-repo/ilab_program_fit_guide.html`

## Context

This fit guide is part of a broader effort to build a **wayfinder tool** for Harvard Innovation Labs — a digital experience that helps students identify the right resource or program for where they are in their innovation journey. The fit framework defined here will inform the decision logic and rubric used to route students to the right program at the right time.
